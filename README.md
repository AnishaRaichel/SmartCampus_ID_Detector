# SmartCampus ID Detector

A Machine Learning project to detect whether a person is wearing an ID card using image classification.

## Models Applied
- Traditional ANN (Feedforward Neural Network)
- Recent CNN (Convolutional Neural Network)

## Steps
1. Add your dataset in /data/with_id and /data/without_id
2. Train models: `python main.py`
3. Test with image: `python test_image.py`
